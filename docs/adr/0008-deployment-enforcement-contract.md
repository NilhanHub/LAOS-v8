# ADR-0008: Deployment and enforcement contract

- Status: **Accepted for the Stage 3 local test profile**

Every claimed capability must map to a mandatory enforcement point owned outside the weaker-agent process. Direct unmediated host operation is unsupported. The final support rows and trusted computing base are frozen only after implementation and adversarial verification.

The implemented boundary and unsupported rows are recorded in `docs/STAGE_3_ENFORCEMENT_CONTRACT.md` and
`PERMISSION_ENFORCEMENT_MATRIX.json`.
