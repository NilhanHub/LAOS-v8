# Architecture decision records

ADRs record decisions that later implementation must follow or formally supersede. A document marked “Accepted” is architectural authority, not proof that the corresponding code has already been implemented.

`ADR_REGISTER.json` is the Stage 1 machine-readable index. ADR-0001 through ADR-0007 are preserved from Stage 0 with their review triggers. ADR-0008 through ADR-0014 are Revision 1.1 proposals and must be accepted by their owning implementation stages before authority depends on them.
